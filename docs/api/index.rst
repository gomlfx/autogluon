API Reference
=============

This API Reference Manual details the usage of functions and objects included in AutoGluon,
describing what they are and what they do. The APIs are grouped into the following categories:

.. container:: cards

   .. card::
      :title: Predictors
      :link: autogluon.predictor.html

      Automatically train/deploy predictors with 1 line of code for text, image, or tabular data.

   .. card::
      :title: Search Space
      :link: autogluon.core.space.html

      Hyperparameter search spaces of candidate hyperparameter values to consider.
